## 1. Core Premise: The Phonosemantic Key
**Phonosemantic Key** = an alignment between **sound (phoneme)** and **meaning (sememe)**. The key idea is that certain phonemes systematically encode semantic roles (existence, breath, force, boundary). This is used to model sacred naming systems (e.g., YHWH) and to create a triadic language structure.

We use a **triadic semantic frame**:
- **Void** (潜/无): negative, hidden, potential
- **Root** (中/本体): stable essence, being
- **Force** (显/有): action, manifestation

Each word is a **3‑slot construct**: `Void–Root–Force`.

## 30. Pronunciation Experiment Protocol (Expanded)
### 30.1 Purpose
This protocol tests **internal coherence** of the breath‑phoneme system rather than
historical reconstruction. The goal is to measure whether a candidate name is:
- **Speakable** (breath‑stable)
- **Rhythmic** (cadence holds across multiple cycles)
- **Intelligible** (phoneme separation is clear)
- **Resonant** (listener reports “sacred” cadence without prompting)

### 30.2 Candidate Sets
Primary candidates (breath‑flow names):
- **Yāū / Iaou / Yawa**
Secondary candidates (command‑phrases):
- **Yaū Lira**
- **Eya Lira**
- **YHWH breath‑sequence** (non‑lexical)

### 30.3 Breath Timing Standard
Use a 4‑phase cycle (seconds are adjustable for the speaker):
- **Inhale (y)**: 2.0s
- **Hold (h)**: 1.0s
- **Exhale (el + phrase)**: 3.0s
- **Void (w)**: 1.5s

### 30.4 Procedure
1. **Warm‑up**: 3 silent cycles (no phonation).
2. **Single‑pass**: speak each candidate once with the 4‑phase cadence.
3. **Loop test**: repeat each candidate 7 cycles without breaking cadence.
4. **Rating** (1–5):
   - **Stability** (breath control)
   - **Clarity** (phoneme boundaries)
   - **Resonance** (felt “rightness” of cadence)

### 30.5 Data Recording
Recommended: record using any phone recorder or command line (`say` on macOS).
Log ratings in a table and compute the mean per candidate.

### 30.6 Expected Pattern (Hypothesis)
- **Yāū / Iaou** should outperform dense‑consonant forms in stability.
- **YHWH breath‑sequence** should score highest in resonance, lowest in lexical clarity.
